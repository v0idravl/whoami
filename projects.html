dagar-red

An ATT&CK-aligned adversary-emulation skill system for Claude Code — 76 technique skills across the full kill chain, each carrying its detection cost and a validation gate. The judgment layer that turns AI into a disciplined red-team operator: it decides which tool call to make next, at which noise tier, and when to stop. /emulate compiles a named actor into an ATT&CK-mapped plan; /innovate synthesizes novel tradecraft when the catalogue runs out. Every engagement's deliverable is improved defense.

sliver-mcp

A Model Context Protocol server for the Sliver C2 framework — exposes the operator surface (listeners, implant/beacon generation, sessions/beacons, execution, file ops, structured handoff) as tools an AI agent can drive, with a passive→green→ yellow→red noise gate. Built for adversary emulation: standing up realistic C2 so defenders can test detection and response.

lab-writeups

Documented lab engagements across Active Directory, Windows, and Linux — enumeration, attack-path reasoning, exploitation evidence, privilege escalation, impact, and remediation notes. Kerberoasting, AS-REP roasting, BloodHound paths, ACL abuse, ADCS ESC chains, DCSync, pass-the-hash, pivoting.

hakiki

A quick-reference knowledge base built from repeated lab work: recon, service enumeration, web exploitation, credentials, privilege escalation, pivoting, Active Directory, ADCS/ACL abuse, and reverse engineering — commands and decision points that keep assessment work consistent.

p0rtix

Scope-aware Python recon and enumeration: port discovery, service classification, web fingerprinting, directory/vhost enumeration, and — in authorized AD labs — credentialed SMB/LDAP enumeration, Kerberoasting, AS-REP roasting, BloodHound collection, and ADCS review, with evidence preserved as raw command logs.

traxerax-lite

Linux security triage: parses auth, nginx, fail2ban, and mail logs into SQLite-backed events and operator reports — the defensive view of attacker methodology and practical infrastructure operation.
